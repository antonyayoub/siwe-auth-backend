# siwe-auth
Backend in NestJS implementing Authentication with SIWE protocol.

## Getting started

### Installation

```bash
$ yarn install
```

### Configure environment

Clone `.env.example` to `.env` and configure following environment variables if needed.


### Running the app

```bash
# start app
$ yarn start
```
